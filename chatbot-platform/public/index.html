How to Embed
Add a chatbot to any website in 2 minutes

Step 1 – Create a chatbot

Go to My Chatbots and click New Chatbot. Fill in the name, instructions, and your OpenAI API key.

Step 2 – Copy the embed snippet

Click Embed on any chatbot card to see its unique snippet.

Step 3 – Paste on your website

Paste the snippet just before </body> on any page.

Platform Where to paste
WordPressAppearance → Theme Editor → footer.php
ShopifyOnline Store → Themes → Edit Code → theme.liquid
WebflowProject Settings → Custom Code → Footer Code
SquarespaceSettings → Advanced → Code Injection → Footer
Plain HTMLBefore </body> in your HTML file

Getting an OpenAI API key

1. Go to platform.openai.com/api-keys
2. Sign in or create a free account
3. Click Create new secret key
4. Copy the key (starts with sk-) and paste it when creating your chatbot
